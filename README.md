# Dữ Liệu Hành Chính Việt Nam

## Giới thiệu
Dự án "Dữ Liệu Hành Chính Việt Nam" nhằm mục đích cung cấp một nền tảng trực tuyến cho việc tiếp cận và phân tích các dữ liệu hành chính tại Việt Nam. Người dùng có thể dễ dàng tìm kiếm và trực quan hóa dữ liệu từ nhiều nguồn khác nhau, giúp nâng cao kiến thức về các vấn đề xã hội và kinh tế hiện nay.

## Tính năng
- Tìm kiếm dữ liệu theo nhiều tiêu chí khác nhau.
- Bộ lọc dữ liệu giúp người dùng chọn lựa thông tin cần thiết.
- Trực quan hóa dữ liệu thông qua biểu đồ và bảng biểu.
- Cung cấp các báo cáo và phân tích tự động.

## Công nghệ sử dụng
- Python
- Flask
- PostgreSQL
- D3.js

## Cài đặt
1. Clone repository này về máy của bạn.
2. Cài đặt tất cả các dependencies.
3. Cấu hình cơ sở dữ liệu và chạy ứng dụng.
